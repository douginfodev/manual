2 - Imagens

2.1 - Adicionar Imagem

1º - Na tela de biblioteca de imagens, clique no botão Adicionar Nova.

2º - Clique no botão Selecionar Arquivo ou arraste aqui uma imagem.

3º - Na tela que abrir, selecione o arquivo que deseja inserir.

4º - Clique no botão selecionar para adicionar a imagem a biblioteca de mídias.

5º - Feito tudo correto, sua imagem aparecerá abaixo na biblioteca.
OBS: Repita estes passos sempre que precisar adicionar imagens ao site.

3 - Imóveis(Property)

3.1 - Visão Geral

1º - Clique na opção Property do menu na barra lateral.

2º - Adicionar Novo imóvel.

3º - Editar Imóvel.

4º - Excluir Imóvel.

3.2 - Adicionar Imóveis

1º - Clique no botão Add New e preencha os dados do imóvel.

2º - Selecione o Tipo de Imóvel.

3º - Adicione a Imagem de Capa.

4º - Clique em Publicar ou Atualizar para salvar.

3.3 - Detalhes do Imóvel

1º - Preencha os campos com as informações do imóvel.

2º - Digite o id da Visão 360.(Caso o imóvel possua a imagem ou video 360º).

OBS: Este id e gerado quando for adicionado a imagem ou video no WPVR.
Se o imóvel não possui imagem ou video 360º, deixe o campo em branco

3.4 - Imóvel - Imagens

1º - Para adicionar a imagem, clique no botãoAdicionar Imagem.

2º - Clique no botão Remover para excluir a imagem

3.5 - Imóvel - Plantas

1º - Para adicionar a imagem da planta, clique no botão Imagem ou no botáo Add.

2º - Repita o passo 1, quantas vezes for necessário, para adicionar novas imagens de plantas.

3.6 - Imóvel - Videos

3.6.1 - Copiar link do video do Youtube

1º - Pesquise o video no YouTube ou Vimeo que deseja adicionar no site.

2º - Clique na opção Compartilhar.

3º - Copie o código do video: EX:https://youtu.be/I2UB3QZ3FF8 e cole no campo youtube da tela abaixo.

3.6.2 - Adicionar o link na página do imóvel

4º - Copie o link para o campo youtube na opção videos.

5º - Clique no botão Publicar ou Atualizar para salvar.

3.7 - Localização

3.7.1 - Copiar Latitude e Longitude do Imóvel (Google Maps)

Clique aqui para acessar o Google Maps.

1º - No site do Google Maps, digite o endereço que deseja buscar a latitude e longitude.

2º - Após buscar o endereç8, Clique em cima do marcador(balão vermelho) para abrir o menu de informações.

3º - Clique em cima dos valores de Latitude e Longitude para copiar os valores.

3.7.2 - Adicionarr Latitude e Longitude no Cadastro do Imóvel

4º - Retorne ao Painel Administrativo, e selecione o imóvel qie deseja incluir os dados.

5º - Nos campos acima, cole (Ctrl + v) os valores de Latitude e Longitude em seus respectivos campos.

4 - TOUR 360 (WPVR)

4.1 - Visão Geral

1º - Clique na opção WP VR do menu na barra lateral.

2º - Botão Nova Tour.

3º - Link Editar Tour.

4º - Link Excluir Tour.

4.2 - Adicionar Imagem 360º

1º - Clique na opção GERAL da barra lateral esquerda.

2º - Digite o nome da tour.

3º - Upload a imagem da tour.

4º - Clique na opção Tour da barra lateral esquerda..

5º - Digite o id da tour disponibilizado na tela de preview()lado direito).

6º - Clique no botão Upload para adicionar a imagem 360º.

7º - Clique no botãoPreview para gerar a tour.

8º - Clique no botão Publicar para salvar a tour.

4.4 - Adicionar Video 360º

1º - Clique np ícone Video na lateral esquerda.

2º - Clique na opção Sim do Habilitar video.

3º - Digite o link do video do youtube / vimeo ou clique no botão Upload.

4º - Clique no botão Preview pata gerar o id do video.

5º - Clique no botão Atualzar pata salvar.

6 - Corretores

6.1 - Visualizar Todos Corretores

1º - Ma barra de Menu Lateral, Selecione a Opção Usuario->Todos Usuariostão Adicionar Nova.

2º - Clque no botáo Adicionar Novo para adicionar novo corretor.

6.2 - Adicionar Corretores

1º - Clique na opção´Usuários na barra de Menu Lateral.

2º - Preencha todos os campos, na tela abaixo.

3º - Na opção Função, selecione Editor.

4º - Clique no botão Adicionar novo usuário para salvar o cadastro;

6.3 - Editar / excluir Corretores

1º - Na tela Visualizar todos Corretores, Usuario->Todos Usuariostão

2º - Para Editar, de um clique em cima do nome do usuário que deseja editar o perfil.

3º - Para Excluir, posicione o ponteiro do mouse, em cima do nome do usuário, e clique no link Excluir.

7.2 - Adicionar(Convidar) Corretor

1º - Clique no botão Administração ma barra de ferramentas superior.

2º - Na barra lateral esquerda, clique em Gestão de Usuários.

3º - Clique no botão Convidar membro.

7.2.1 - Envisr convite so corretor

1º - Digite o email do corretor que deseja enviar o convite.
OBS: Caso deseja enviar convites para mais corretores, clique no botão + Adicionar outro.

3º - Clique no botão Enviar convite para enviar email com o convite para o corretor.

7.2.3 - Email e criação de conta de acesso ao chat-online

1º - Com o email automático, que o corretor recebeu, clique no botãocreate account para ser redirecionado a tela de criaçã´da conta. OBS: Caso deseja enviar convites para mais corretores, clique no botão + Adicionar outro.

2º - Na tela de criaçãó de conta, preencha o campo Insira seu nome.
OBS: Este nome é o que será mostrado no chat-online quando estiver conversando com o cliente.

3º - Confirme o email. Não necessita ser alterado.

4º - Crie uma senha de acesso.

5º - Clique no botão Registra-se para ter acesso ao painel admistrativo do chat-online;

7.4 - Accessar Sala do chat-online

1º - Todos os clientes que estao online e aguardando uma resposta.
Dè dois clique, em cima do identificador do cliente, para abrir a sala do chat.

7.5 - Criar ticket de atendimento ao vhat-online

1º - Na tela do painel administratico, clique no botão(icone)Caixa de Entrada para visualizar as mensagens.

2º - De dois cliques em cima do nome do contato para ter acesso a conversa.

3º - Clique no botãoConverter em ticket.

4º - Preencha os dados acima e clique no botãoCriar.

5º - Clique no botãoX para fechar a tela.
